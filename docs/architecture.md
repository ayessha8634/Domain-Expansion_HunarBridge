# HunarBridge Architecture

## Overview
HunarBridge is a visual-first platform designed to empower artisans in informal settlements. 

## Structure
- `frontend/`: React + Vite + TypeScript application using Tailwind CSS v4.
- `backend/`: Firebase Functions handling business logic and AI assistant integration.
- `database/`: Firestore security rules and index definitions.
- `docs/`: Technical and product documentation.

## Tech Stack
- **Frontend**: React, Framer Motion, Lucide, Leaflet, i18next.
- **Backend**: Firebase Functions (Node.js).
- **Database**: Google Cloud Firestore.
- **Storage**: Firebase Storage (or Cloudinary).
- **AI**: Web Speech API for voice-to-text.
